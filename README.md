## 📋 Installation

### Step 1: Clone Repository
```bash
git clone https://github.com/addotwumnana-eng/mt5-gold-scalper-ea.git
```

### Step 2: Copy EA Files
1. **For BTC EA (24/7)**:
   - Copy `BTCScalperEA.mq5` to `MQL5/Experts`

2. **For GOLD EA (Sessions)**:
   - Copy `GoldScalperEA.mq5` to `MQL5/Experts`

### Step 3: Compile
1. Open MetaTrader 5
2. Go to **File → Open Data Folder**
3. Navigate to **MQL5/Experts**
4. Right-click `.mq5` files → **Compile** ✅

### Step 4: Attach to Chart
1. Open appropriate chart (BTCUSD or XAUUSD, M5 timeframe)
2. Drag EA from Navigator → Chart
3. Configure parameters
4. Click **OK** ✅

## 🔧 Troubleshooting

### EA Not Trading?
- ✅ Check if EA enabled (green face icon)
- ✅ Verify trading allowed (Tools → Options)
- ✅ Check magic number (should be unique)
- ✅ Review logs for errors

### Poor Performance?
- ✅ Adjust MA periods
- ✅ Change trailing stop settings
- ✅ Update profit/stop levels
- ✅ Check if in correct session/hours

### Pending Orders Not Filling?
- ✅ Check spreads (too wide = no fills)
- ✅ Verify order distance is realistic
- ✅ Ensure account has margin
- ✅ Check pending order expiry
